**Status**: `decision needed`

**Proposal**: `package.json` version field + git tags. `v1.0.0` is the first public release tag. DNS resolution is explicitly not part of the tag criteria ŌĆö the tag records content/code readiness, not domain availability.

**Version semantics for a content site** (post-1.0):
