This documentation explains Anthropic's data usage policies for consumer Claude products (Free, Pro, Max). User data is used for model training only with explicit consent, during safety reviews, or when users opt into programs like the Trusted Tester Program. Incognito chats are explicitly excluded from model training even when Model Improvement is enabled.
