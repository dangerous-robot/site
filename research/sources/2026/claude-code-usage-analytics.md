This documentation covers the Claude Code usage analytics feature available to Team, Enterprise, and API Console users. It details organization-level metrics such as lines of code accepted and suggestion accept rates, as well as user-level metrics and optional GitHub contribution metrics for measuring engineering velocity. The page includes setup instructions, metric explanations, and frequently asked questions.
