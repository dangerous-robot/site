This support documentation details Anthropic's External Researcher Access Program, which offers $1000 in free Claude API credits to researchers focusing on AI safety and alignment. Applications are reviewed monthly on the first Monday, and the program includes an FAQ addressing common questions about access restrictions, usage policies, and eligibility requirements.
