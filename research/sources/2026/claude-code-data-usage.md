This documentation page details Anthropic's data handling practices for Claude Code across different user tiers. It covers data training policies distinguishing between consumer and commercial users, retention periods ranging from 30 days to 5 years depending on account type and preferences, and technical details about data encryption and telemetry services. The page also explains data flows for both local and cloud-based Claude Code execution.
