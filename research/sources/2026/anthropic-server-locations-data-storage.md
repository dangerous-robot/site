This documentation is part of Anthropic's Privacy Center and addresses server location and data hosting for commercial Claude products including Claude Enterprise and the Anthropic API. The article clarifies that while data processing may occur across multiple regions, persistent data storage defaults to US locations unless otherwise specified by customer agreement.
