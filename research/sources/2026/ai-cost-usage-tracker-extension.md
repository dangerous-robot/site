This is the GitHub repository for the AI Cost & Usage Tracker extension. The project provides token counting, cost estimation, energy and carbon impact tracking, and decision-time guidance across multiple AI platforms. The extension stores all usage data locally in browser storage and includes optional features like budget alerts, anomaly detection, and model recommendations.
