OpenAI redesigned ChatGPT's model selection interface to present three subscription-dependent tiers (Instant, Thinking, Pro) instead of individual model names, with dropdown access to specific versions. Users can also configure the Auto function to switch between tiers based on task complexity. The change addresses longstanding user frustration with the routing system's opacity and perceived cost-cutting bias toward cheaper models.
