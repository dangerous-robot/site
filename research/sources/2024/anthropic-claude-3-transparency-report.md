This page is part of Stanford's Foundation Model Transparency Index (May 2024) and provides a detailed assessment of Anthropic's Claude 3 model against a set of transparency indicators. The report covers upstream factors (data practices), model-level details (architecture, frameworks, hardware), and downstream elements (capabilities, limitations, risks, and mitigations). The assessment shows scores across multiple evaluation categories with justifications and references to Anthropic's published documentation.
