- asking Llama 3.3 70B to write a typical email uses approximately 0.1841 Watt-hours
  — equivalent to running a microwave for 0.12 seconds

The tool works with Chat UI, an open source front end for models like Meta's Llama 3.3 70B and Google's Gemma 3, showing energy consumption in Watt-hours or Joules and comparing usage to common household appliances. It aims to increase awareness about the environmental impact of AI usage as growing demand for AI technologies is expected to drive electricity needs to new heights.
