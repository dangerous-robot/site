Anthropic's official documentation confirms that Claude does use user data for model training, but only with user consent. According to 'Is my data used for model training?', regular chats can be used for improvement if users explicitly choose to allow it, while Incognito chats are excluded. The 'Updates to Consumer Terms and Privacy Policy' announcement states that as of October 8, 2025, Anthropic is implementing an opt-in system where users can voluntarily share conversation data for model training and safety improvements, with a 5-year retention period. WIRED's reporting notes this represents a policy shift, as Anthropic was previously "one of the few major AI companies not automatically using user conversations for training." The sources establish that Claude does use user data for training purposes, though users retain control through opt-in/opt-out mechanisms. Commercial products like Claude for Work and the API operate under different terms and are excluded from this consumer data usage policy.
