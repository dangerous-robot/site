Anthropic provides detailed public documentation of the Claude models it operates. The “Introducing the next generation of Claude” announcement lists the three Claude 3 variants—Haiku, Sonnet, and Opus—along with their capabilities, context windows, and pricing (Introducing the next generation of Claude). A separate “Models overview” page enumerates the current Claude 4 models (Opus 4.7, Sonnet 4.6, Haiku 4.5) with IDs, token limits, and usage guidance (Models overview). For developers, the “List Models” API reference describes an endpoint that returns the exact set of models available to an account, making the information queryable programmatically (List Models). Finally, Anthropic’s “Model Report” offers model‑cards for each version, describing training data, safety evaluations, and release dates (Model Report). Across these sources, Anthropic consistently discloses which Claude models exist and their specifications, confirming that Claude does disclose the models it uses.
