The claim is contradicted by available evidence. "Llama and ChatGPT Are Not Open-Source" explicitly states that ChatGPT scores poorly on openness criteria despite any open-source claims. The IEEE Spectrum research found that "few ostensibly open-source LLMs live up to the openness claim," specifically including ChatGPT in this assessment. While "ChatGPT creator OpenAI is getting ready to release an open-source AI model, report says" indicates OpenAI may release an open-source model in the future, this confirms that ChatGPT currently does not use open-source models.
