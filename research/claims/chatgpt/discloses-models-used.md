OpenAI’s public documentation lists the full catalogue of its models, including the latest GPT‑5.5 series, indicating that the company shares detailed model information (All models, 2026). Independent technical articles also state explicitly that the ChatGPT product is built on specific model versions—GeeksforGeeks notes that ChatGPT is based on GPT‑3.5, and Scalable Path describes the same architecture (ChatGPT's Architecture, 2024; How Does ChatGPT Actually Work? An ML Engineer Explains, 2023). Together, these sources confirm that OpenAI publicly discloses the underlying models used by ChatGPT.
