GreenPT explicitly lists the open-source models used in its documentation pages. The developer and API documentation (Source 5) states that the platform uses 'open-source models', while the features page (Source 6) specifies the use of the Deepgram Nova-3 model. However, the specific detailed model names are not enumerated. A third-party article on the Green Web podcast roundup (Source 7) mentions GreenPT as using Mistral mini and GPT-OSS 120B, adding independent confirmation. While third-party sources (Source 8) highlight model specifically deployed models, the primary first-party disclosures (Source 1, 2, and 3) lack a comprehensive model-by-model list. Thus, while GreenPT openly states its use of open-source models, the exact detailed list of models is partially transparent, and parsing customer-facing materials requires deductive work for full clarity.
